# Student Management System

A console-based Student Management System developed using Core Java.

## Features

- Add Student
- View Students
- Search Student
- Update Marks
- Delete Student
- Calculate Average Marks
- Find Highest Scorer

## Technologies Used

- Core Java
- Arrays
- Methods
- Loops
- Conditional Statements
- Scanner Class

## Concepts Covered

- Variables
- Data Types
- Arrays
- Methods
- Loops
- Switch Case
- User Input

## How to Run

1. Clone the repository.
2. Open the project in Eclipse or IntelliJ IDEA.
3. Compile and run `StudentManagementSystem.java`.
